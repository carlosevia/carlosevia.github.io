--- layout: default title: "How to help?" ---

The best ways to contribute are by finding and reporting bugs, participating in community support, and improving the code.

Reporting bugs

We use GitHub as our issue tracker. There you can submit bug reports and make feature suggestions.

Community support

Answer DITA-OT related questions at DITA-OT Users, DITA users group, or StackOverflow.

Patch the code

We use GitHub as our version control. You can contribute fixes to bugs or new features by sending pull requests at GitHub.

Developers and vendors using the toolkit are encouraged to join the DITA-OT developer mailing list. This list is used to discuss upcoming changes that may impact developers or vendors.
